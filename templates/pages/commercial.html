Dental X-ray Disease Detection

Python Computer Vision YOLO Machine Learning HTML/CSS

Developed an intelligent system for the automatic analysis of dental X-rays. This project features a local web application where users can upload raw dental X-ray images. Once uploaded, a custom-trained YOLO object detection model processes the image to precisely identify, locate, and graphically highlight diseased or unhealthy teeth with bounding boxes. This automated visual analysis acts as a powerful, real-time diagnostic assistant for dental professionals, significantly speeding up the initial review process and improving accuracy.

Personal RAG Assistant for TikTok

Python RAG Architecture LLMs SQLite JSON

A smart Telegram bot powered by Retrieval-Augmented Generation (RAG) created specifically to assist a TikTok content creator. The bot acts as a personal manager: it can answer account-specific questions, brainstorm new ideas, and help plan weekly content schedules. It leverages a local SQLite database (knowledge.db) to securely store conversation history and structured data, alongside JSON files (account_info.json, your_content.json) for flexible configuration and content management.
